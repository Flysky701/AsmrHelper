# 2026-03-23 工作日志

## 项目：ASMR音频汉化工具

### 对话要点
用户希望创建一个可以对音声进行音频插入/汉化翻译功能的项目。

### 讨论的技术问题
1. 立体声处理 - AI处理立体音频
2. 效果音频与空间感 - 保持ASMR沉浸感
3. 音频识别 - Whisper识别低语、气声
4. 文本更正与逻辑通顺 - NLP后处理
5. 人声分离与保留 - UVR5等模型
6. 音色与情感保持 - TTS+声音转换

### 推荐改进方案
- 基础版：基于 GalTransl-for-ASMR 或 VoiceTransl 扩展
- 进阶版：融合 Faster Whisper 与空间音频库
- 专业版：全链路自定义工具链

### 核心开源项目
- GalTransl-for-ASMR（ASMR字幕翻译）
- VoiceTransl（AI字幕软件）
- Faster Whisper TransWithAI ChickenRice（针对ASMR优化）
- GPT-SoVITS（语音克隆/TTS）
- Buzz（Whisper桌面端）

### 参考的视频配音项目
- **pyvideotrans**：功能全面的视频翻译配音工具，支持Faster-Whisper、GPT-SoVITS语音克隆、多角色配音、GUI界面
- **Bluez-Dubbing**：模块化微服务架构，REST API+CLI+Web UI，支持VAD音频对齐
- **KrillinAI**：一键启动，支持100+语言，多平台输出（抖音/B站等），Go编写
